00230 /*      
00231  * A parametric curve is a function r(u): [0,1] -> R^d that described a curve in a d-dimensional
00232  * space. The function r(u) is implemented as the pure virtual make_r(u) and must be defined in
00233  * the derived classes. Given r(u) all the other geometric properties are computed here. 
00234  *
00235  * Notice that the first and second parametric derivatives make_ru() and make_ruu() are implemented 
00236  * by numerical differentiation but they are also declared as virtual so that more efficient and 
00237  * precise implementation can be included in derived classes (eg splines).
00238  *
00239  * The class is equipped with a Gauss Quadrature class to perform integration along the curve.
00240  *
00241  * The optimized calculation tree is shown below:
00242  *
00243  * \verbatim
00244  * 
00245  * make_ruu()       make_ru()      make_r()        (CRTP level: implemented in derived class)
00246  *     ^                ^              ^
00247  *     |                |              |                                
00248  *     |            make_j()           |
00249  *     |                ^              |
00250  *     |                |              |
00251  *     |            make_rl()          |                        
00252  *     |                ^              |
00253  *     |                |              |              
00254  *     +----------------+
00255  *     |
00256  * make_ju()
00257  *     ^
00258  *     |
00259  * make_rll()
00260  *     ^
00261  *     |
00262  * make_kappa()
00263  *
00264  * \endverbatim
00265  */
