Detailed Description

template<typename Derived>
class model::StaticID< Derived >

A class template that implements a counter of the number of instances of Derived type that are created at runtime. It also provides a unique increasing static identifier for each instance. 

 #include <model/Utilities/StaticID.h>
 
 class MyClass : public model::StaticID<MyClass>{};

 int main(){

 MyClass a;             // with default constructor
 MyClass b(a);  // with copy constructor

 std::cout<<a.sID<<std::endl;
 std::cout<<b.sID<<std::endl;
 }
